## 🚩 项目概览
### 配置文件结构
```bash
EFI/
├── BOOT/
│   └── BOOTx64.efi          # UEFI 引导文件
└── OC/
    ├── ACPI/                # DSDT/SSDT 补丁
    ├── Kexts/               # 核心驱动
    ├── Drivers/             # OpenCore 驱动
    ├── config.plist         # 主配置文件
    └── ...                  # 其他资源文件
```

- **仅适配 ASUS FL8700JP (X509JP) 机型**，其他设备不保证兼容性。
- **不包含 macOS 镜像**，需自行准备安装文件。
- 安装前请备份数据，**操作风险自负**。

## ⚙️ 安装指南
### 1. BIOS 设置
```bash
(Secure Boot) ➔ 关闭 (Disabled)
快速启动(Fast Boot) ➔ 关闭 (Disabled)
DVMT 预分配(DVMT Pre-Allocated) ➔ 64M
```

### 2. 安装步骤
```bash
1. 下载 EFI 文件 ➔ 解压至 USB 驱动器
2. 制作 macOS 安装盘 ➔ 推荐使用 BalenaEtcher
3. 调整启动顺序 ➔ 从 USB 启动
4. 运行 OpenCore ➔ 选择 "Install macOS"
5. 完成安装后 ➔ 迁移 EFI 至系统盘
```

### 3. SMBIOS 配置
```bash
推荐机型：MacBookPro16,2
工具：OpenCore Configurator 或 OCAT
注意：必须生成唯一三码！
```

## ❓ 常见问题
<details>
  <summary><b>Q: HDMI 无信号？</b></summary>
  Ice Lake 处理器已移除 HDMI 控制器支持，建议使用 Type-C 转 DP 接口。
</details>

<details>
  <summary><b>Q: 触控板偶发失灵？</b></summary>
  ELAN1200 固件限制，暂时通过重启或睡眠唤醒恢复。
</details>

<details>
  <summary><b>Q: 如何切换默认系统？</b></summary>
  OpenCore 界面按 Ctrl+Enter 或系统设置 ➔ 启动磁盘。
</details>
