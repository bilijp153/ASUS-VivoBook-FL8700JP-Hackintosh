 🛠️ 安装教程
 三步快速部署
1. BIOS设置
   markdown
   -   关闭Secure Boot
   -   设置DVMT为64M
   -   启用AHCI模式

2. EFI配置
   bash
   git clone https://github.com/bilijp153/ASUS-VivoBook-FL8700JP-Hackintosh
   cp -R EFI /Volumes/ESP/

3. 系统安装
   markdown
   1. 使用OpenCore引导启动
   2. 选择「Install macOS」
   3. 完成基础配置后迁移EFI

 ❓ 常见问题
<details>
<summary>💡 高频问题解答</summary>

Q：HDMI无法输出怎么办？  
A：Ice Lake处理器原生不支持HDMI输出，建议使用Type-C转DP

Q：触控板间歇性失灵？  
A：这是ELAN1200驱动已知问题，尝试：
bash
sudo kextunload VoodooI2C.kext
sudo kextload VoodooI2C.kext

Q：Windows/Mac双系统切换无声？  
A：在Windows中安装微软官方HDA驱动
</details>
